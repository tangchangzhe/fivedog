# 掌控 - 小狗陪你做计划 🐕

一只可爱的小狗帮你管理待办事项！

## 创意概念

- 🐕 与小狗聊天，告诉它你今天想做什么
- 📝 点击小狗，它会吐出纸条（待办事项）
- ✅ 完成任务后，把纸条喂给小狗
- ⭐ 小狗吃掉纸条后会拉出星星，收集到星星瓶里！

## 功能

- 左侧聊天区：与AI小狗对话，收集待办事项
- 右侧互动区：可爱的小狗 + 星星瓶
- 纸条系统：任务以纸条形式展示
- 完成奖励：星星收集系统

## 部署

```bash
npm install
node server.js
```

服务运行在 `http://localhost:3000`

## 技术栈

- Node.js + Express
- TailwindCSS
- DeepSeek AI API
- 纯CSS动画
